Type E
미니멀 텍스트 탭

👍 장점

  • 상단 한 줄에 탭이 정리되어 시선 이동이 가장 짧고 직관적
  • 굵기와 밑줄로 선택 상태가 명확해 한 눈에 현재 위치 파악
  • 라디오 버튼이나 아이콘 없어 해석 부담이 가장 적음
  • 화면 여백이 넓어 고급스럽고 세련된 분위기 연출
  • 검색 아이콘이 입력 필드 안에 있어 공간 효율 최고
  • 불필요한 시각 요소 제거로 콘텐츠에 집중 유도

👎 단점

  • 비활성 탭이 회색 텍스트로만 표시되어 구분이 다소 약함
  • 터치 영역이 텍스트 크기에 한정되어 오조작 가능성
  • 시니어나 디지털 문해력이 낮은 사용자에게 다소 어려울 수 있음
추천
Type A
탭 버튼 스타일

👍 장점

  • 박스 형태로 터치 영역이 넓어 모바일 조작성 우수
  • 선택된 탭의 배경색으로 현재 위치를 즉시 파악 가능
  • 대부분의 앱에서 사용하는 패턴으로 학습 비용 제로
  • 시니어 사용자도 쉽게 이해할 수 있는 명확한 구조

👎 단점

  • 2010년대 초반 iOS 스타일로 트렌디하지 않음
  • 박스 테두리가 두꺼워 무거운 느낌
  • LINE의 젊은 브랜드 이미지와 다소 불일치
추천
Type B
라디오 버튼 스타일

👍 장점

  • 라디오 버튼으로 선택 상태를 명확히 인지 가능
  • 검색 아이콘이 각 입력 필드의 목적을 직관적으로 전달
  • 깔끔한 라운드 디자인으로 모던하고 세련된 느낌
  • 여백과 타이포그래피 균형이 우수해 가독성 최상
  • 기능성과 심미성을 모두 만족하는 밸런스형 디자인

👎 단점

  • 라디오 버튼이 탭 전환보다는 폼 입력처럼 보일 수 있음
  • 아이콘과 버튼 요소가 많아 시각적으로 다소 복잡함
Type C
왼쪽 타이틀 + 소형 탭

👍 장점

  • 타이틀과 탭을 분리해 정보 계층이 명확함
  • 상단 공간을 압축해 콘텐츠 영역 확보에 유리
  • 우측 소형 탭이 부가 기능처럼 보여 심리적 부담 감소
  • 한 눈에 페이지 맥락을 파악하기 쉬운 구조

👎 단점

  • 탭 버튼이 너무 작아서 터치 실수 가능성 높음 (최소 44px 권장)
  • "Find friends" 타이틀과 탭이 같은 줄에 있어 혼란스러움
  • 작은 탭이 중요한 기능임을 인지하기 어려움
  • 첫 사용자가 탭 전환 기능을 발견하지 못할 위험
  • 좌우 정렬이 달라 시선 흐름이 불안정함
Type D
아이콘 + 텍스트 탭

👍 장점

  • 아이콘이 각 탭의 기능을 시각적으로 명확히 설명
  • 아이콘+텍스트 조합으로 다국어 사용자에게 유리
  • 시니어나 디지털 문해력이 낮은 사용자도 쉽게 이해
  • 브랜드 아이덴티티를 강화할 수 있는 커스텀 아이콘 활용 가능

👎 단점

  • 아이콘 2개가 탭 영역을 과도하게 차지해 무거워 보임
  • 텍스트만으로 충분한데 아이콘이 중복 정보 제공
  • 아이콘 의미가 명확하지 않으면 오히려 혼란 야기
  • 수평 공간을 많이 사용해 콘텐츠 영역 압박
